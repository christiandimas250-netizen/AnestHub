# AnestHub
AnestHub integra herramientas esenciales para la práctica anestésica diaria, incluyendo cálculos hemodinámicos, gasométricos y ventilatorios, estimación de requerimientos transfusionales, dosificación de fármacos inductores y calculadora TCI manual. Diseñada para brindar apoyo clínico rápido, preciso y accesible en el entorno perioperatorio.
